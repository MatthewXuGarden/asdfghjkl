Report settings

Path:

, Report management, Setup page

 

 

 

Purpose

To create HACCP, log or real-time report templates that are printed/exported by the user or automatically for scheduled activities.

Description

Reports can be type:

  • Instant: the printed values will be those read at printout For variables of any type.

  • Daily: printed values are those read in the last 24 hours Only for log or HACCP variables.

  • Weekly: printed values are those read in the last 7 days Only for log or HACCP variables.

  • Customized: printed values are those read in the interval set when printing is requested. Only for log or HACCP variables and only for reports printed by the user.

You can only choose between regular sampling frequency (5 s., 30 s.) and customized. In this case you can indicate up to four different times when variable values are read. For Daily, Weekly, Custom report only; use the Frequency, Custom option. 

Possible output formats include:

  • PDF

  • Excel

  • HTML

What to do first

Set which and how device variables are recorded for each device (see chapter Device configuration: logs).

What to do next

Only for scheduled activity reports, set a report print action in the scheduled activity (see chapter Scheduled activities)

 

A report template cannot be deleted if used as a printout action in a scheduled activity.

To select several elements from a list:

  • hold down the CTRL key and click on the single elements

or:

  • hold down the SHIFT key and click on the first and last element to be selected

  • Select the interval, layout, sampling frequency, output format and whether HACCP

  • Select Devices: the list of system devices appears in the left column

  • Select the device: its log or HACCP variables appear in the right column

  • Select all the variables to be added to the report and click

    • or double-click a single variable

  • Click to add the report template to the list
