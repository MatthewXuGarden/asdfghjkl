Site calendar: introduction

Path:

Click and select Plant Calendar

Access:

 

 

Purpose

To schedule integer, analog and digital variable settings on a weekly basis.

Manually set digital, integer and analog values on variables in the same group.

Enable/disable group scheduling.

Description

  1. Select variables of the same type to be used and group them in categories (i.e.: defrost, heating).

  2. Select the devices that have these variables and group them (i.e.: fridge banks/compressors).

  3. In the Main page, set the Standard weekly calendar setting the times of the day where certain values are set on variables in each category (i.e.: at the hottest times of the day, reduce the temperature set point for more exposed fridge banks).

  4. Set an Exception calendar for exceptional and repetitive dates where different times and values are set (i.e.: holidays).

Linked pages

Click on the link to open the explanation on the relevant pages:

Main: to graphically create intervals or weekly regulation times and set exceptions.

Groups: to group similar devices in groups.

Categories: to group similar variables in categories.
