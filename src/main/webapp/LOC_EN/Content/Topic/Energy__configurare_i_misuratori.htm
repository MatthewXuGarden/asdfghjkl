Energy: Setting up meters

To set consumption controls:

  1. Check for or create energy meters.

  2. Select the monitoring settings (fixed cost or time slot).

  3. For fixed cost monitoring, set cost calculation and CO2 emission parameters.

  4. For time slot monitoring, set cost categories.

  5. Create groups of meters for homogeneous utility groups (i.e.: refrigerator counters) or for site areas (i.e.: warehouses).

Check for or create energy meters.

  1. In Meter Models, check whether energy meters are already listed: click to add a new meter, click to edit an existent one or click delete it.

To add a new meter model:

  1. Enter the new model name in Meter Model.

  2. Click Select.

  3. Link a meter listed in Settings, Device model.

  4. Select the power and energy variables listed in Variable.

  5. Click Set to link them to the meter model.

Setting fixed cost monitoring

  1. In the Settings page select Fixed Cost and set cost calculation and CO2 emission parameters.

  2. Adding data for report headings and footers.

Setting time slot monitoring

  1. In the Settings page select Time Slot and set cost calculation and CO2 emission parameters.
  2. Enter the relevant cost per kWh for one or more cost categories.

  1. Double-click on a time slot (slot 01 in the example): the hours in the days of the typical week and cost categories set for each hour appear.

  1. To set a cost category for an entire day, click Set row and select the cost category. To set a cost category for a single hour but for every day of the week, click Set column and select the cost category. To set a cost category on every day of the week, click on the bottom right.

  2. To create time exceptions, double click Exceptions: a new week appears.

  3. Click Exception days to select the days on which the exceptional week are applied.

To copy/paste one week's settings, click to copy and click to paste.

Creating/editing groups

  1. In Groups, enter the group name (i.e.: refrigerator counter) and tick the enable flag.

  2. In Meter Model, select the meter model to be added to the group.

  3. Use the arrows to add or delete meters to/from the group.

  1. Click to add a group, click to edit it or click to delete it.

  2. Click Save.

Replacing a meter requires a series of steps to save data read until the meter is replaced.

During maintenance on utilities connected to a meter, we suggest you disable the entire group and not just the single meter. In fact, values read by a meter during maintenance could invalidate all group data.
