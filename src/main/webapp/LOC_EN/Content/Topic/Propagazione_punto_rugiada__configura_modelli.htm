Dew point broadcast: template setup

Path:

, Dew point broadcast, Template setup page

 

 

 

Purpose

Customizes variable propagation modes from a Master to its Slaves.

Description

-

 

Only valid for templates that support Master-Slave architecture.

Action

Setting up a template for dew point broadcast.

  • Select a Master-Slave type template (i.e.: MPXPRO)

  • If the template is used as a Master, indicate for the three source variables:

Variables to be read

Variable name (i.e.: Probe B2)

Alarm variables

Linked alarm variable (i.e.: Probe B2 fault)

Min
Max

Variable validity interval If the value exceeds the limits, it is not propagated

Safe

Safety value, to be propagated if the source value exceeds the limits.

  • If the template is used as a Slave, indicate the target variables respectively:

Variables to be written

Variable name (i.e.: Probe B2)

  • Indicated the variables to be monitored, if required:

Control variables

Variable name (i.e.: Glass temperature)
