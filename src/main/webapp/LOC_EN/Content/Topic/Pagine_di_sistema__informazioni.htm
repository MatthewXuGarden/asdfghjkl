System pages: Information

Path:

, System pages, Information page

 

, Wizard, Registration page

 

Purpose

Starting the product.

Start the engine when it is stopped.

Identifies PlantVisorPRO Locale so it can be recognized by RemotePRO or customer care (via FTP).

Description

PlantVisorPRO is license protected software Once installed, the trial version can be used for a maximum of 30 days. It must be "activated" before the end of the trial period. Therefore, you must receive an activation code from CAREL that must be entered in the System Setup page.

 

The product freezes and no longer records data at the end of the trial period. Data acquired in the trial period are saved in the database. To release and activate the product, contact customer care.

Site ID data must be communicated to RemotePRO managers.

All product functions are enabled during the trial period. The activation code must be entered to activate the product and optional functions (found on the CAREL website via Internet connection). Do not update the product to the same version. Updates to subsequent product versions do not include trial periods.

Identification data is required by RemotePRO software and customer care (access via FTP) but are not required for the user to open PlantVisorPRO Locale from a browser.

Action

  1. Obtaining an activation code

    • Enter the serial number indicated on the CAREL label on the bottom of PlantVisorPRO Embedded:

    • In the CAREL web site PlantVisorPRO page, click Request PlantVisorPRO activation code

    • In the Installer and User section, enter the mandatory data and carefully read privacy information.

    • In Data required for code issue enter:

      • Serial number

      • MAC address indicated in the PlantVisorPRO Locale information page and listed on the outer label.

    • Confirm having read privacy information

    • In System, enter data useful to business statistics

    • Click Next, check data and click Activation code: if data is correct, an activation code will appear

  2. Activating the installed product

    • In the System setup menu Information page, enter the serial number and activation code

    • Click : the product is licensed and can be used

  3. Identifying the site

    • Entering data:

    Name

    System description (i.e.: Supermarket refrigeration) The entered name appears in the login page.

    Telephone

    Phone number dialed by RemotePRO to connect to this site via modem.

    Ident

    System identification number different from all other systems managed by the same RemotePRO.

    Necessary for data alignment

    password

    Site access password via RemotePRO

    Necessary for data alignment

    Must contain at least six characters. Case sensitive
