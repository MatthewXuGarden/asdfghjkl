HVAC Smart Start: dashboard

Path:

, HVAC Smart Start, Dashboard page

 

 

 

Purpose

This page shows the HVAC Smart Start real-time trend and status for the current week.

Description

-

What to do first

Set operations. See the Settings page.

Page description

Page area descriptions

Area Description

1

Area with HVAC Smart Start real-time data and status in the current week:

Following is a description of some data:

  • Estimated time ON: time calculated for the climate control system to turn on.
  • Estimated time OFF: time calculated for the climate control system to turn off. The icon appears if the function is still calculating the estimated time.
  • Time balance: difference between the early climate control system off minutes (compared to building closing time) and early climate control system on minutes (compared to building opening time) calculated for the last seven days (i.e.: 5 h 7 min (green) = savings, -1h 20 min (red)= loss).
  • from: date of last reset, from which the Time balance is calculated

2

The current weekly trend can be displayed in two ways:

  • °C: indoor and outdoor temperature logged on each day when the climate control system turned on and off
  • Minutes:
    • Red: minutes running until the set point is reached at building opening time
    • Green: minutes before the climate control system turns off to keep the set point at building closing

Action

View on/off savings trend
  • In the Algorithm name summary table, select the customization

Other actions

Start HVAC Smart Start

Stop HVAC Smart Start

Update HVAC Smart Start

Reset the Time balance meter and update the date.
