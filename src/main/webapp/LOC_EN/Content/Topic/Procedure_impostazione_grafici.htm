Setting log variables for graphs and reports

  1. Setting variable recording methods

    • In Device configuration, Log config   set graph, HACCP or log report variables (see chapter Device configuration: logs)

Customizing graphs

  1. Customize HACCP and log graphs on the device level

    • Select a device "template" to start settings

    • In Device details, HACCP graph or Log graph set the HACCP or log graph layout, variable aspects and limits (see chapter Monitoring: HACCP and log graphs)

  2. Propagate the settings to similar devices

    • In Site configuration, Copy config, select the devices where the graph settings are copied to (see chapter Configuration propagation)

  3. Customize graphs to compare group variables

    • In Global, <groupName>, HACCP graph or Log graph, set HACCP or log graph layouts used on the group level (see chapter Monitoring: HACCP and log graphs) 

  4. Customize graphs to compare variables on the global level

    • In Global, HACCP graph or Log graph, set HACCP or log graph layouts used on the global level (see chapter Monitoring: HACCP and log graphs) 

  5. Propagate the entire configuration to other users

    • In Site configuration, Copy config, select the profile from which the configuration is to be copied and target profiles (see chapter Configuration propagation)

Creating a single graph

  1. Creating a single graph

    • In Device details, HACCP graphs or Log graphs set the graph to be displayed (see chapter Monitoring: HACCP and log graphs)
