PlantVisorPRO Shell: system user administration

Path:

PlantVisorPRO Shell, click User administration

 

 

 

Purpose

Set the passwords for system accounts:

  • User

  • Maintenance technician

  • Installer

Description

-

 

WRITE DOWN YOUR PASSWORD! If you no longer have a password, contact CAREL customer care.

Use the default password indicated in the Instruction sheet included in packaging or download it from the CAREL website for first login.

System accounts are only used to access PlantVisorPRO Shell System administration and User administration functions. To set program accounts for PlantVisorPRO Locale users, refer to chapter Program users: configuration.

Action

Changing system account passwords

  • Enter the previous password and a new password at least 8 characters long and confirm

  • Wait for the confirmation message
