Print commissioning report

Path:

Click and select Commissioning report

 

Installers, Users

 

Purpose

Exports all real-time values of all site parameters in Excel format (input device variables) typical at the end of site configuration.

Print in .PDF/export some site parameters selected by the user to Excel format.

Description

The user can set up a customized report, selecting which parameters should be added and print/export their values.

The user can open a default report created by CAREL and print/export values.

If a default report is opened, customized report settings are lost.

The user can export all site parameter values.

Output formats:

  • .CSV (Excel)

  • .PDF (print only)

Action

Creating and printing a custom report with data from a single device.

  • Select Devices: the list of system devices appears in the left column

  • Select the device: its settings appear in the right column

  • Hold down the CTRL key and click on all the settings to add to the report and click

    • or double click each single setting

  • Click Save.

  • Select the export format (.CSV or .PDF)

  • Click Print to export to the selected format.

Creating and printing a custom report with data from a several devices.

  • Select device templates and select a type: the list of devices of that type appears in the left column and its settings appear in the one on the right

  • Hold down the CTRL key and click on all the settings to add to the report and click

    • or double click each single setting

  • Click Save.

  • Select the export format (.CSV or .PDF)

  • Click Print to export to the selected format.

Printing the commissioning report set by CAREL.

  • Click Default to load the default report

If the default report is opened, the settings in the displayed custom report are lost.

  • Select the export format (.CSV or .PDF)

  • Click Print to print/export the report.

Exporting real-time values for all site parameters to an Excel file

  • Select .CSV format to enable the export button

  • Click Export all to export to the selected format.

Settings in the displayed custom report are saved.
