Report printout

Path:

, Report management, Print/export page

 

 

 

Purpose

To print/export an HACCP or log report in the format indicated in the template.

Description

If report format is PDF, it can be immediately printed.

Data will be treated as follows based on the type of report:

  • Instant: the printed values will be those read at printout

  • Daily: printed values are those read in the 24 hours prior to the indicated date

  • Weekly: printed values are those read in the 7 days prior to the indicated date

  • Customized: printed values are those read in the indicated period

What to do first

The installer must set report templates (see chapter Report: report configuration).

Select the printer where reports will be printed and enable it (see Printer notification channel setup)

 

If no variables appear for the selected device, ask the Installer to program log or HACCP variables for that device.

If reports are printed at set intervals (i.e.: at end of the day), ask the Installer to program automatic report printout.

Action

Printing reports

  • Select the report template

  • For all reports, except instant, select the required dates.

  • Click : the report appears in a new window. Print or save.

Exporting a report

  • Select the report template

  • For all reports, except instant, select the required dates.

  • Click : the report appears in a new window. Print or save.
