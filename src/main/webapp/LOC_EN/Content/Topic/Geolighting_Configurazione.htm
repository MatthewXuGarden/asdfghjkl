GeoLighting: light on/off setup

Path:

, GeoLigting, Configuration page

 

 

 

Purpose

This page lets you set GeoLighting parameters to automatically turn lights on and off inside the building.

Description

-

What to do first

-

What to do next

Start GeoLighting in the Dashboard page

 

You can edit settings also when GeoLighting is running.

Setting descriptions

Latitude

Longitude

Building latitude and longitude

Day variable

Variable associated with indoor/outdoor light on/off command

Night variable

Variable associated with indoor/outdoor light on/off command

Action

Set light on/off control
  • Manually enter data in the Latitude and Longitude text boxes.
  • Click Select: the Variable list appears
  • Select Devices to view the list of all devices,
  • or select Device templates to view the template list: the list of the selected device variables appears
  • Select the variable associated with the required controls
  • Click Set
  • Click Save.
  • Open the Dashboard page to start GeoLighting
