Device monitoring: main page

Path:

, Global, double-click the device,  Main page

 

, Devices,click a device

 

 

 

Purpose

Displays the physical or logical device status and its variables. Starts/stops device functions as if working on its terminal and sets service parameters.

Description

Various page sections show daily status, outputs, probes and alarms. Read only and write variables are only displayed upon request.

 

The page layout can be customized (see chapter Device configuration: monitoring page).

Action

Controlling the device

  • Use the control buttons to run functions.

Setting write variables

  • Click to expand the variable table.

  • If necessary, click to load current values from the device.

  • Edit values.

  • Click to send data to the device.

Selecting another physical or logical device

  • Use the summary box at the top left to select a different device

Other actions

Double-clicking an alarm row

Opening the Alarm details page.

Reading all current variable values from the field.

See the Technical sheet for PlantVisorPRO Locale acquisition time.

Writes the values in the entry boxes on the field. If the box is empty, the value remains constant.

Active alarm list print preview (HTML).
