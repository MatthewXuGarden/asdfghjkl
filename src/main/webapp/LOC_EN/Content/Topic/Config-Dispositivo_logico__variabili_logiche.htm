Logical device setup: logical variable creation

Path:

, Logical device creation, Logical variables page

 

 

 

Purpose

Monitors complex situations (through alarms and graphs) that require logical-mathematic calculations.

Example

Define a logical variable that represents the sum of several refrigeration unit absorption.

Description

Calculations may be made between two or more physical variables from different devices or between a variable and a constant.

What to do next

Include the logical variables in a logical device to treat it just like any physical variable.

Action

Creating a logical variable

  • Add the description

  • Select the variable type and device

  • Select the operator

  • Select a variable from the list

  • Click to include it in the list of "operand" variables.

  • Repeat for other variables

Editing a logical variable

  • Double-click a variable row

Other actions
