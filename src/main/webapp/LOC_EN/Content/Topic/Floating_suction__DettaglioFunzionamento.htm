FSP: operating details

From the Main page, click on the rack [1] to view details or edit algorithm parameters.

Floating Suction algorithm details

Floating Suction details help to understand which showcase is running a regulation request.

For example, if all rack showcases have duty cycles under the maximum duty cycle set for the showcase (or are in regulation request conditions for less than the set maximum), the algorithm lowers the rack set point value.

If at least one showcase has a duty cycle over the set maximum, on the other hand, the algorithm increases the rack set point value.

The page displays the list of parameters used and duty cycle values for each showcase:

Area Description

1

Viewed rack details.

2

Parameter list: some have the current value, others the set value.

3

Showcase duty cycle values.

4

Reset: resets duty cycles on all showcases calculated until that point.

5

RST: resets the duty cycle on a showcase calculated until that point.

 For a description of rack parameters, see Algorithm parameters.

Possible actions

Press RST to reset the duty cycle for a showcase calculated to that point, for example, if the set duty cycle maximum changed (see Customizing parametersi).

Press RESET to reset all duty cycles on showcases calculated to that point, for example, if rack parameters changed (see Customizing parameters).

Smooth lines algorithm details

The page displays the values and status of the rack and linked showcases in the various set sampling periods:

Area Description

1

Viewed rack details.

2

Parameter list: some have the current value, others the set value.

3

Rack set point values in the various set sampling periods.

4

Showcase status in the various set sampling periods.

5

Number of showcases that block Smooth lines and total number of yellow, orange and red leds in the last sampling period ("T" column).

6

: shows/hides from the fourth to the tenth samplings before the last one.

7

RST: deletes completed sampling.

Operating principle

The algorithm compares the amount of samples acquired in a certain status (i.e..: yellow led = warning) with set parameters (i.e.: maximum three yellow leds in a sample). Based on the comparison, it changes the rack set point as follows:

When at last sample [T]… The rack set point it…

All leds are green

reduced

Number of yellow or orange leds < Maximum set number

Number of yellow or orange leds >= Maximum set number

maintained

Number of red leds < Maximum set number

Number of red leds >= Maximum set number

increased

At least one led N/A

reduced

Possible actions

Press to show/hide samples prior to the last three.

Press RST to delete completed sample: all leds turn black.
