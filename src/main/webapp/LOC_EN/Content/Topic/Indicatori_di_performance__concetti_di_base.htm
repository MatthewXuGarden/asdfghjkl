Performance indicators: basic concepts

The module checks the effective device regulation capacity, indicating:

  • when variable data is over and under a limit

  • when a value remains under the minimum limit or over the maximum limit for an excessive period of time

Calculations are made considering the period set by the user that can take into account any defrost or non regulation periods (i.e.: when the device is off).

Devices can be grouped. The variables to be checked are set for each device template.

Report fields

Total time

Analysis period actually used for calculations.

Total time = Period - (Sum of defrost time) - (Sum of non regulation time).

Time under the limit

Variable duration under the Min limit.

% time
under limit

% duration under the limit. The field has a  blue  background if the value is under the Min % limit. 

Time over the limit

Variable duration over the Max limit.

% time
over limit

% duration over the limit. The field has a  red  background if the value is over the Max % limit. 

Medium

Variable average calculated over Total time.

Defrost time

Total defrost period duration.

% defrost time

% defrost periods of Total time.

ON time

Total regulation period duration.

% duty cycle

% regulation periods of Total time.

% duty cycle = ON time / (Total time - Defrost time)
