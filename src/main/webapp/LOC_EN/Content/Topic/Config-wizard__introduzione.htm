Installer help file (setup) > Setup Wizard > Introduction

Setup wizard

Path:

Click and select Wizard quick configuration

Access:

Installers, Users

 

Purpose

Quickly setup the site.

Description

Guided site setup procedure. The page can always be opened from the menu.

Linked pages

Click on the link to open the explanation on the relevant pages:

1 - Registration: to register the product and set site data. Similar to the System page: information

2 - Line configuration: to setup the line. Similar to the Line detail page.  

3 - Parameter settings: to set main device parameters.

4 - Alarm management: to set notification recipients (e-mail, fax, text message (SMS)) or relays to be started in the event of an alarm.

5 - HACCP report: to set the acquisition time for HACCP values to be printed.

6 - guardianPRO: to start guardianPRO and enable/disable notifications. Similar to the System page: guardianPRO setup.

For advanced setup

For advanced PlantVisorPRO settings, open the advanced setup pages:

1 - Registration

Subsequent changes may be made from the System: information page under the Site configuration function.

2 - Line setup

Subsequent changes may be made from the Line detail page under the Site configuration, line configuration function

3 - Set. parameters

Subsequent parameter changes may only be made on the single device level, in page Device monitoring: parameter settings under the Device monitoring function.

4 - Alarm management

Subsequent changes to alarm, notification and alarm action management can be made in their setup pages respectively:

  • Site setup: alarm priorities under the Site setup function.

  • Alarm and event activities function

  • Testing notification channels function

5 - HACCP report

Subsequent changes can be made in the Report setup page, under the Report function.

6 - guardian PRO

Subsequent changes can be made in the System: guardianPRO setup page under the System pages function.
