Floating Suction algorithm parameters

Following are the more significant parameters:

Parameter

Description

Default

Maximum set point

Maximum value available for the Set Point variable.

-

Gradient

It sets how the set point variable value will be increased or decreased following duty cycle calculations on showcases.

-

Time window for DC calculations

Indicates how long the data corresponding to the solenoid on each individual showcase is saved in the memory. The duty cycle will be calculated based on these data; once the window is full, each new value will overwrite the oldest value.

The value may vary from 10 to 180 minutes.

12 minutes

DC calculation frequency

Indicates how often the duty cycle must be calculated for rack showcases and, consequently, the set point changed if necessary.

The value may vary from 10 to 180 minutes.

30 minutes

Maximum failed connection time

Maximum percentage of failed acquisitions for each showcase associated with the rack If this is exceeded, the showcase is considered offline.

30%

Maximum number of showcases not connected

If when the duty cycle is calculated the showcases that are offline exceed this number, the set point is not changed on the rack.

Rack and linked showcase status

Led color

Description of the rack or showcase status

(green).

Showcase OK. The showcase is correctly operating from the Smooth lines algorithm standpoint: the rack set point can be increased.

(yellow)

First warning: the showcase has switched from OK (green led) to NOT OK.

(orange)

Second warning: the showcase remains in NOT OK status for 2 consecutive samples (the led was yellow at the last sample).

(red).

Third warning: the showcase remains in NOT OK status for 3 consecutive samples (the led was orange at the last sample).

(black)

Offline: the plug-in cannot read the showcase or rack set point value.

N/A

The showcase does not support the Smooth lines algorithm: the plug-in cannot read the showcase set point variable value.

If N/A status persists for a showcase, it should be eliminated from the group. In fact, the rack is forced to constantly work in safety conditions, at minimum set point, when the Smooth lines algorithm is not supported.
