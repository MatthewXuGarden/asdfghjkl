FSP: customizing rack and showcase parameters

Floating Suction Pressure Control is supplied with standard parameters that manage plug-in operations. Nonetheless, some of the compressor rack parameters or the maximum duty cycle of a showcase can be changed, for example to adjust the algorithm to the specific installation.

If the Smooth lines algorithm is enabled, the following can be set and edited at any time: time interval between samples and number of altered states (Yellow, Orange, Red) when the plug-in intervenes.

The custom parameters are always saved in the database, even after disabling the compressor rack (see Configuring Floating Suction Pressure Control).

After having modified the rack or showcase parameters, always re-start the plug-in

Floating Suction algorithm parameters

In some cases, rack parameters must be changed if, for example, there are several showcases in a rack and the maximum number of offline showcases needs to be increased. Alternatively, if the installation has network problems, the maximum offline time may need to be increased.

If, looking at the data in the Details page and the recorded duty cycle values are higher than the set duty cycle maximum, single showcase operations must be checked and the set duty cycle maximum may need to be increased.

Following is what the Parameters page looks like:

 

Area Description

1

Rack whose parameters are to be edited.

2

Rack parameters used by the plug-in.

3

Maximum duty cycle admitted on all showcases.

4

Maximum duty cycle admitted on a single showcase.

Possible actions

  1. Modify the parameters and click to save the changes.
  2. Stop and restart the control to make the changes effective.
