Installer

Installation

  • Install PlantVisorPRO setting up peripherals and communications with RemotePRO.

  • In PlantVisorPRO, set system users, those who have access to low level functions.

Configuration

  • Configure the system using the standards device settings or previously installed system settings.

  • Set guardianPRO to control computer and PlantVisorPRO functions

  • Define PlantVisorPRO Locale users, those who have access to setup, monitoring and supervision functions.

  • Define periodic reports, graphs and logs to be transferred to RemotePRO.

  • Set scheduled activities and alarm/event activities.

Maintenance

  • Perform site maintenance, finding the cause of the faults indicated by PlantVisorPRO, checking unit performance and adding new devices to the configuration
