Purpose

Disables guardianPRO for a certain period of time (i.e.: system maintenance time band).

Enables/disables notifications to listed contacts.

Indicates guardianPRO notification recipients.

(optional) Setup active alarm controls and variable checks.

Description

Refer to chapter guardianPRO: basic concepts, to see why active alarm control and variable checks are beneficial.

What to do first

Set the maximum alarm management time.

Set up the notification channels to be used.

 

guardianPRO notifications warns the user of malfunctions and faults that jeopardize entire system operations. If notifications are disabled, guardianPRO will continue its control activity but can only run events or reboot the system where applicable.

In general, system status controls are always on even if guardianPRO is disabled for several days. Temporarily disabling guardianPRO effects guardianPRO notifications following negative results: check results are saved in the guardianPRO log but new events, notifications or reboots are not run.

Action

Selecting guardianPRO notification recipients

  • Click Enable notifications.

  • Select an addressee from the address book

  • Click to include it in the list of guardianPRO recipients

  • Repeat for other addressees

(optional) Selecting variables whose variations are controlled

  • Click Check probe variation to be warned when, for a certain amount of time, the indicated variables always read the same value.

  • Selecting a device

  • Selecting a variable

  • Click to include it in the list of monitored variables.

  • Repeat for other variables

  • Indicate the Number of consecutive constant values to signal an error.

Other actions

Common used buttons

guardianPRO disabled for days

Indicate the days it should be off.

Enable control on active alarms.

Indicates when any alarms are not managed by PlantVisorPRO Locale within a set time band.
