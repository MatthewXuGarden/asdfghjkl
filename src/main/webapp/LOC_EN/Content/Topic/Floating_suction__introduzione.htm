FSP - Suction pressure optimization

Path:

Click and select FSP - Suction pressure optimization

Access:

 

 

Purpose

To reduce showcase energy consumption.

Description

To do this, it periodically analyzes showcase operations connected to the enabled refrigeration racks and automatically changes the rack set point based on:

  • percent work values (duty cycle) calculated for each showcase (Floating Suction algorithm)
  • showcase status at sampling (Smooth lines algorithm)

This minimizes the delivery of liquid refrigerant and consequently reduces power consumption in the medium and long term.

Of the devices present in the installation, Floating Suction Pressure Control only involves those that manage showcases or refrigeration units and, for these devices only displays pressure control parameters. The parameters of the two algorithms are already set in the factory, however they can be customized based on specific features of the installation.

Procedure

The plug-in must be first configured and then activated. To do this, proceed as follows:

  1. Select which racks are to be controlled, link them to showcases, select and set up the control algorithm (see Setting up Floating Suction).

  2. Launch the plug-in (see Start up and monitoring).

  3. Keep rack set point variations under control to regulate algorithm parameters if necessary (see Operating details and Customizing rack and showcase parameters).

Once the plug-in has started, the following operations are possible:

  • Enable/disable a rack for control.

  • Reset duty cycle calculations for all showcases or a single showcase (see Operating details).

  • Edit rack parameters and maximum duty cycle for each showcase (see Customizing rack and showcase parameters)

  • Enable/disable the Smooth lines algorithm.

  • Edit rack parameters and conditions that, if met, launch the plug-in (see Customizing rack and showcase parameters).
