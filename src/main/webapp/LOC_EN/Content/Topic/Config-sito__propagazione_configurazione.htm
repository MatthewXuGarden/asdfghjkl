Site configuration: configuration propagation

Path:

, Site setup, Copy setup page

 

 

 

Purpose

Quickly sets up devices of the same type starting from a device "template" setup. Setups made by a user with a certain user profile can be copied for graphs.

Import the device setup from other sites.

Export the device setup to be used on other sites (for example, copying it to a USB key).

Description

After selecting the device "template", the list of devices of the same type which are setup targets is updated. Setup parts to be copied reflect the corresponding Device setup pages (see chapter Device setup).

 

PlantVisorPRO Locale does not check data consistency in the setups to be propagated. The developer must check completeness and validity.

To import/export an entire site setup or all site notification rules, refer to System pages, System see chapter System: system status).

Devices must be enabled but can be disconnected.

MPXPRO and MPX devices : this function propagates the setup on all devices, regardless of the field Master-Slave architecture.

  • Select a sample device or in Import setup, select a custom device from another site.

Variable display
