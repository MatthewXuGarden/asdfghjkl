Activity: activation rules

Path:

, Alarm and event management, Rules page

 

, Activity scheduler, Rules page

 

Purpose

Set the rule (group of actions, time bands and conditions) that define the activity.

Enable/disable actions associated with the rule.

Description

-

 

The rule description must be as descriptive as possible since it is the text sent to action recipients.

To disable alarm and event activities from a device, see the device setup Main page.

Action

Creating a rule for scheduled activities

  • Enter the rule name (i.e.: "Communication channel check"). This is the name of the rule that will be written in the fax, text message, e-mail and remote supervisor text.

  • Select the time band when the activity is enabled

  • Select the action

  • Enable the rule

  • Click

Creating a rule for alarm/event activities

  • Enter the rule name (i.e.: "Electrical fault service call")

  • Select the alarm or event condition

  • Select the time band when the activity is enabled 

  • Select the action

  • Enable the rule

  • Click
