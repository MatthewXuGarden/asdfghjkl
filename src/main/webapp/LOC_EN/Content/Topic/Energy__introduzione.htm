Energy: consumption analysis

Path:

Click and select Energy - Analyses and energy reports

Access:

 

 

Purpose

To monitor electrical consumption and relevant costs, CO2 emission and optimize consumption,

Description

Energy analyzes real-time consumption values in kW and kWh returned by field meters (power analyzers, energy meters or simply suitably programmed pCO devices) and links them to the relevant costs, even variable, by time slot and period.

Procedure

The function must first be set and enabled.

To set the function:

  1. Set groups with meters and the site meter and set calculation parameters (see Setting up meters).

  2. Program utility start/stop using the Plant calendar function.

  3. Starting Energy (see Monitoring consumption).

Once the function has started, the following operations are possible:

  1. Monitor real-time consumption (see Monitoring consumption)

  2. Analyze past consumption (see Analyzing consumption in the period) and export them to a .csv file (see Exporting hourly, daily or monthly consumption)

  3. Stopping the function.
