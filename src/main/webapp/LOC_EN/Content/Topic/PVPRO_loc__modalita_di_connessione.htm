Connection modes

There are three ways to connect to PlantVisorPRO Locale:

  • from RemotePRO, if you know the local ID and with a dedicated connection (modem) or via Internet.

  • from Internet Explorer, if you know the static IP address of the computer where PlantVisorPRO Locale is installed.

  • via VPN connection.

To communicate with PlantVisorPRO Locale from RemotePRO, refer to the RemotePRO help file.

To view PlantVisorPRO Locale pages from a browser, enter the following in the machine's static IP address bar:

https://xxx.xxx.xxx.xxx/PlantVisorPRO/

A window asking for a certificate opens in Internet Explorer version 7. In you browser, disable option Warn about certificate address mismatch.
