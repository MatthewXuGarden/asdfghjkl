Smart Night Purge: setup

Path:

, Smart Night Purge, Setup page

 

 

 

Purpose

This page lets you set Smart Night Purge parameters. Enter:

  1. Master conditioner and any Slave conditioner data
  2. indoor and outdoor temperatures required for Smart Night Purge operations
  3. other data required to optimize consumption

Description

-

What to do first

-

What to do next

Start Smart Night Purge from the Dashboard page

 

Settings can also be edited while Smart Night Purge is running.

Setting descriptions

Parameter Description

MASTER - Unit status

Variable that manages Master conditioner on/off status

MASTER - Unit ON/OFF control

Variable that manages Master conditioner on/off control

Number of Slaves

Number of Slave conditioners (from 1 to 3)

SLAVE 1/2/3 – Unit status 

Variable that manages Slave conditioner 1, 2, 3 on/off status.

SLAVE 1/2/3 - Unit ON/OFF control

Variable that manages Slave conditioner 1, 2, 3 on/off control

Internal temperature

Variables that read internal temperature

Outside temperature

Variable that reads outside temperature

Internal humidity (*)

Percent internal humidity (manually entered or selected by the variable)

Outdoor humidity (*)

Percent outdoor humidity (manually entered or selected by the variable)

Temperature set point

Internal temperature set point (manually entered or selected by the variable)

Humidity set point (*)

Internal humidity set point (manually entered or selected by the variable)

Temperature set point differential

Margin of tolerance from set point (deadband)

Humidity set point differential 

Margin of tolerance from set point (deadband)

Maximum unit on time 

Maximum free cooling length, from sunrise

Fan consumption (free-cooling mode)

Master and relevant Slave conditioning unit energy consumption in free cooling mode (inferred from factory default value)

Fan capacity

Fan capacity (inferred from factory default value)

Plug-in deployment start date

Date of action/service start in Smart Night Purge

Plug-in deployment end date

Date of action/service end in Smart Night Purge

Free-cooling end time action

Maximum free cooling end time, meaning sunrise (manually enter the time or have it automatically read by GeoLighting)

 

Note (*): in lack of an indoor/outdoor humidity probe, set all three fields to 50. This enables free cooling for temperature values and not enthalpy.

Action

Customize free cooling for building areas
  • Click Add: the fields are set to default values for a new customization
  • Enter the custom name in the Algorithm name text box
  • Set free cooling
  • Click Save: the name of the new algorithm appears in theAlgorithm list summary box

Warning: customizations that are not saved are automatically deleted when the function is closed.
Set night free cooling
  • Click Select: the Variable list appears
  • Select Devices to view the list of all devices,
  • or select Device templates to view the template list: the list of the selected device variables appears
  • Select the variable associated with the required controls
  • Click Set
  • Click Save.
  • Open the Dashboard page: to start Smart Night Purge

Other actions

Adding a new algorithm customization.

Deleting the selected customization.
