Procedura per completare installazione di PlantVisorPRO

L'utente responsabile della messa in rete del computer deve possedere tutti i requisiti tecnici di installatore di sistema.

PlantVisorPRO Locale deve essere un sistema dedicato alla supervisione. Leggere attentamente i punti 6. Garanzie e vostri diritti e 8. Rischi a vostro carico della Licenza d'uso (in Gestione sistema, Licenza d'uso).

Operazioni protette da password di sistema.

  1. Completare l'installazione di PlantVisorPRO

  2. Collegare PlantVisorPRO a RemotePRO via modem

  3. Collegare PlantVisorPRO a RemotePRO via Web

Completare installazione PlantVisorPRO

  1. Collegare il computer

    • Seguire le indicazioni del foglio istruzioni allegate per il collegamento del computer e la prima installazione

    • All'accensione compare il pannello PlantVisorPRO Shell con le funzioni riservate agli installatori (vedi capitolo PlantVisorPRO Shell)

  2. Modificare le password di sistema per Installatori e Manutentori

    • In PlantVisorPRO Shell, Gestione utenti provvedere a inserire delle nuove password per prevenire usi impropri del computer (vedi capitolo Gestione utenti sistema).

  3. Leggere la licenza d'uso

    • In PlantVisorPRO Shell, Gestione computer, Licenza d'usoleggere il documento (vedi capitolo Gestione utenti sistema).

  4. Attivare il prodotto

    • Seguire le indicazioni nella pagina Sistema: informazioni.

  5. Configurare le periferiche

    • In PlantVisorPRO Shell, Gestione computer, Configura modem procedere alla configurazione dei modem interno e aggiuntivo (vedi Gestione del computer)

    • In PlantVisorPRO Shell, Gestione computer, Configura stampanti e fax, procedere alla configurazione (vedi Gestione del computer)

  6. Definire le modalità di connessione per ogni periferica

    • In Configurazione periferiche I/O, Fax definire le modalità di chiamata e un numero telefonico per fare successivamente il test (vedi capitolo Configurazione canali notifiche: fax).

    • In Configurazione periferiche I/O, SMS definire le modalità di chiamata e un numero telefonico per fare successivamente il test (vedi capitolo Configurazione canali notifiche: SMS).

    • In Configurazione periferiche I/O, e-mail definire le modalità di invio e un indirizzo di posta elettronica per fare successivamente il test (vedi capitolo Configurazione canali notifiche: e-mail).

  7. Fare un refresh delle periferiche

    • In Pagine di sistema, Sistema fare il refresh delle periferiche (vedi capitolo Stato del sistema).

  8. Provare le periferiche

    • In Test periferiche I/O, fare il test per ogni canale di comunicazione (vedi capitolo Test canali di notifica)

Collegare PlantVisorPRO al Remoto via modem

  1. Configurare il modem

    • In PlantVisorPRO Shell, Gestione computer, Configura modem procedere alla configurazione del modem (vedi Gestione del sistema)

  2. Riavviare il computer

    • In PlantVisorPRO Shell, Gestione computer, Riavvia il sistema.

  3. Identificare il sistema

    • In Configurazione sito, Sito identificare il sito per il collegamento da RemotePRO (vedi capitolo Pagine di sistema: informazioni)

  4. Definire le modalità di connessione

    • In Configurazione periferiche I/O, RAS definire le modalità di connessione e il numero telefonico di RemotePRO (vedi capitolo Configurazione canali notifiche: chiamate a remoto).

  5. Provare la connessione

    • In Test periferiche I/O, fare il test della connessione verso RemotePRO

Collegare PlantVisorPRO al Remoto via Web

  1. Mettere in rete il computer

    • In PlantVisorPRO Shell, Gestione computer, Confuigurazione indirizzo IP, configurare correttamente i campi

  2. Ottenere un IP statico

    • Chiedere all'amministratore della rete l'IP statico da assegnare a PlantVisorPRO Locale.

  3. Riavviare il computer

    • In PlantVisorPRO Shell, Gestione computer, fare clic su Riavvia il sistema.

  4. Identificare il sistema

    • In Configurazione sito, Sito identificare il sito per il collegamento da RemotePRO (vedi capitolo Pagine di sistema: informazioni)

  5. (opzionale) Abilitare accesso FTP per l'assistenza

    • In Pagine di sistema, Sistema attivare il server FTP (vedi capitolo Stato del sistema).

  6. Controllare la connessione

    • Da RemotePRO aggiungere i dati del nuovo sito e verificare che avvenga la connessione
