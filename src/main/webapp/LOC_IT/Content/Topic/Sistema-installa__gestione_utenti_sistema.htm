PlantVisorPRO Shell: gestione utenti di sistema

Percorso:

PlantVisorPRO Shell, fare clic su Gestione utenti

 

 

 

Scopo

Definire le password per gli account di sistema:

  • Utente

  • Manutentore

  • Installatore

Descrizione

-

 

ANNOTATEVI LE PASSWORD! Se non si è più in possesso delle password di accesso sarà necessario contattare l'assistenza CAREL.

Per il primo accesso usare le password di default indicate nel Foglio istruzioni incluso nell'imballo oppure scaricabile dal sito CAREL).

Gli account di sistema servono solo per accedere alle funzioni Gestione sistema e Gestione utenti di PlantVisorPRO Shell. Per configurare gli account di programma, ovvero per definire gli utenti che avranno accesso a PlantVisorPRO Locale, fare riferimento al capitolo Utenti di programma: configurazione.

Azioni

Cambiare la password degli account di sistema

  • Inserire la password precedente e una nuova password di almeno otto caratteri e confermare

  • Attendere il messaggio di conferma
