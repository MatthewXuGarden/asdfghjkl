PlantVisorPRO Shell: backup e ripristino

Percorso:

PlantVisorPRO Shell, fare clic su Backup

 

 

 

Scopo

Lanciare, schedulare o ripristinare un backup completo.

Fare il backup, o ripristinare il backup della configurazione. 

Fare il backup per l'assistenza.

Descrizione

Indicare la cartella dove salvare il file.

Il backup/ripristino configurazione è analogo a quello che si può lanciare dalla pagina Sistema.Le differenze del backup da PlantVisorPRO Shell, rispetto a quello da pagina Sistema, sono:

  • Può essere lanciato solo da un operatore che accede alla macchina e non da Web

  • Esegue il backup non solo della configurazione ma anche del software

  • Il backup completo può essere schedulato a intervalli regolari

  • Il ripristino può importare anche backup di versioni precedenti di software

 

Durante un backup il sistema di supervisione resta spento. Il backup di un sito complesso (es.: 150 dispositivi per un anno di funzionamento) può richiedere circa 15 minuti. Il sistema si riavvia automaticamente alla conclusione del backup.

Si consiglia vivamente di fare il backup su cartelle esterne alla macchina.

Nel ripristinare un backup prestare la massima attenzione alla versione del backup e della versione attualmente installata.

In tutti i supervisori aggiornati alla versione 1.5, la schedulazione di default è fissata al primo martedì del mese alle ore 6.00.

Il comando Unzip backup è usato solo dall'assistenza remota.

Azioni

Schedulare un backup completo

  • In Schedule selezionare la frequenza o la data del backup e la cartella esterna di destinazione

  • Fare clic su Apply

Lanciare un backup

  • In Backup selezionare la cartella esterna di destinazione

  • Fare clic sul tipo di backup da lanciare:

    • Backup completo: copia totale della macchina, usato per creare un clone di una macchina

    • Backup configurazione: fa il backup del sito e regole, senza i dati storici.

    • Backup di servizio: riservato all'assistenza.

Ripristinare i dati salvati in un backup

  • In Recovery scegliere il file di backup

  • Controllare la corrispondenza delle versioni

  • Fare clic su Site Recovery se si sta ripristinando un backup completo

    • oppure fare clic su Configuration Recovery se si sta ripristinando un backup della configurazione
